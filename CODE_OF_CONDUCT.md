# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

## 我们的标准

有助于创造积极环境的行为：
- 对新手友好、有耐心
- 提供建设性反馈
- 承认错误并从中学习

不可接受的行为：
- 骚扰、侮辱、贬低
- 发布他人隐私信息
- 其他在专业环境中不合理的行为

## 执行

项目维护者负责澄清和执行行为标准，并有权删除、编辑或拒绝不符合本行为准则的评论、提交、代码等。

如有骚扰或其他不可接受的行为，请通过 GitHub Issues 联系维护者。

## 来源

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.1。
